# Mac "Zoom" Button Performance Issue

On Mac, when pressing the green "zoom" button in the application's toolbar, Mac will override the resolution settings to a much higher resolution than intended.

<figure><img src="../.gitbook/assets/image.png" alt="Mac application window buttons"><figcaption></figcaption></figure>

If you press the green "zoom" button to maximize the window and experience performance issues:

1. Navigate to the main menu and select Settings
2. Select the Resolution dropdown
3. Select a new Resolution from the dropdown
4. Reselect your desired Resolution from the dropdown

This will reset the resolution in the simulator.
